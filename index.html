포맷·유형·제작자·제작요소 설정은 이 브라우저에만 즉시 저장됩니다. 모두에게 공유하려면 [설정 내보내기]로 YYMMDD_채널명_세팅.json 파일을 받아 GitHub 저장소에 업로드하세요. 업로드된 최신 세팅 파일은 다음 접속 시 자동으로 반영됩니다.
채널 데이터 업로드
엑셀 파일을 드래그하거나 클릭해서 업로드 (여러 채널 동시 가능, 3년치 여러 파일 누적 가능)
파일명 형식: YYMMDD_채널명_포맷.xlsx (예: 260618_서든어택_숏폼.xlsx / 260618_서든어택_롱폼.xlsx)
자동 포맷: 파일명 끝의 _숏폼 또는 _롱폼을 읽어서 모든 행의 포맷을 자동 지정합니다.
자동 유형: 제목에 BGM이 포함되면 유형이 자동으로 BGM으로, 감정표현이 포함되면 업데이트로 자동 지정됩니다. (업로드 후 팝오버에서 언제든 수동 변경 가능)
세팅 공유 파일: YYMMDD_채널명_세팅.json 형식으로 올리면 포맷/유형/제작자/제작요소 설정이 모든 사용자에게 공유됩니다.
